# Listas-AED
Códigos da resolução das listas de exercícios da cadeira de Algoritmos e Estruturas de Dados CIn UFPE

## Aviso
Alguns desses códigos podem estar incompletos pois não consegui resolver o problema a tempo.
Então, é bom ver se está tudo direitinho com eles.

## Listas
Cada lista tinha um foco de resolução no assuto que estava sendo dado nas semanas em que estava aberta.
Elas insentivavam a implementar algumas estruturas e algoritmos para solucionar o problema.
Assim, no meio desses códigos estão a implementação de alguns algoritmos e estruturas conhecidas
e que podem ser bem úteis em algumas situações.
